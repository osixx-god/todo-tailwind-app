# Todo App 📝

A clean, responsive Todo app built with React and Tailwind CSS.

## 🔗 Live Demo
[View Live](https://todo-tailwind-app.vercel.app)

## ✨ Features
- Add and delete tasks
- Mark tasks as complete
- Filter by All / Active / Completed
- Light and dark theme toggle
- Persistent storage with localStorage
- Fully responsive design

## 🛠 Built With
- React
- Tailwind CSS v4
- Vite
